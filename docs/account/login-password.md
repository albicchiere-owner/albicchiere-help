# Login Issues and Password

## Forgot Your Password?

1. On the login screen, select **"Forgot Password"**
2. Enter your registered email address
3. Check your inbox for the reset link
4. Create a new password following the required criteria
5. Confirm and log in

## Can't Log In?

- Make sure you're using the **correct email address** for your account
- Check that Caps Lock is off
- Try resetting your password (see above)
- If you signed up with Google or Apple, use those login buttons instead

## Change Your Password

1. Go to **Your Profile**
2. Select **"Settings and Privacy"**
3. Choose **"Change Password"**
4. Enter your current and new password
5. Save
