# Check Bag and Quality Score

## Check Bag

Verify your wine's authenticity and NFC tag status:

1. Select the **Wine Glass icon**
2. Press **Check**
3. Scan the bag or box tag
4. Wait for scan completion
5. A **green checkmark** confirms authenticity

## Quality Score

The Quality Score monitors your wine's freshness and condition over time. It is calculated automatically based on:

- Time since opening
- Storage temperature
- Preservation conditions

A higher score means fresher wine. Check it anytime from the wine detail page.

## Handle Check Bag Errors

| Error | Solution |
|-------|----------|
| NFC Tag Not Detected | Ensure tag and phone are in close contact, try again |
| Invalid NFC Tag | Verify you're scanning the correct tag on the bag or box |
| Counterfeit Smart Bag | Only use official Albicchiere Smart Bags |
