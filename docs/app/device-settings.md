# Device Settings and Technical Info

## Access Device Settings

**If a Wine Bag is installed:**
1. Go to Explore
2. Select the dispenser card
3. Tap the **settings icon** (top right)

**If no Wine Bag is present:**
1. Go to Explore
2. Tap the dispenser card directly

## What You Can Do in Device Settings

- View technical specifications
- Rename your dispenser
- Change the room
- Set cooling mode
- Manage parental controls
- Reset to factory settings

## Technical Information

Scroll to the bottom of Device Settings to find:

| Data | Description |
|------|-------------|
| Model Number | Hardware version |
| Serial Number | Unique device ID |
| MAC Address | Network identifier |
| Wi-Fi Status | Connection state |
| Firmware Version | Current software version |

This information is useful when contacting support.

## Factory Reset

Go to **Device Settings → Restore Factory Settings** and confirm. This will erase all local settings on the device.

:::caution
A factory reset cannot be undone. The device will need to be set up again from scratch.
:::
