# How to Add a Smart Wine Bag

## Steps

1. Tap the **Wine Glass icon** on the home screen
2. Select **"Add Smart Wine Bag"**
3. Albi will start the **wine check process**

### If successful:
- Tap **"Add to Cantina"** to save the wine to your collection

### If unsuccessful:
- Check the error message shown
- Make sure the bag is correctly positioned in Albi (see [Bag Positioning](../smart-dispenser/smart-bag-positioning))
- Try again or add the wine manually using [Label Scan](label-scan)
