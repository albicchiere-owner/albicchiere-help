# Find and Add Wine from the Catalog

## Search by Name

1. Go to the **Search section**
2. Select **"Browse the Catalog"**
3. Enter the wine name in the search field
4. Select the wine and **Add to Cantina**

## Search with Filters

1. Select **"Filters"**
2. Choose your preferences:
   - **Category** (red, white, rosé, sparkling…)
   - **Vintage** (year)
   - **Format** (bottle or Smart Wine Bag)
   - **Mood** (passionate, meditative, carefree…)
   - **Food pairing**

## Adding a Wine Not in the Database

If you can't find your wine, add it manually to your cantina. Albicchiere's team will review and add it to the database for all users.
