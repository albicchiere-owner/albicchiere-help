# First Setup

You only need to set up Albi once.

1. Power on Albi
2. Follow the **on-screen instructions** to connect Albi to your Wi-Fi network
3. Open the Albicchiere app and **pair the device**
4. That's it — Albi will handle everything from now on

:::info
You will **not** need to repeat this process every time you use Albi. After the initial setup, it connects automatically.
:::
