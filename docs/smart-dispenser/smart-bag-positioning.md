# How to Position the Smart Wine Bag

Proper positioning of the Smart Wine Bag is essential for automatic wine recognition and optimal dispensing.

## Steps

1. **Remove the cap** from the Smart Wine Bag (see [How to Remove the Cap](remove-cap))
2. **Orient the bag** with the spout facing downward
3. **Slide the bag** into the compartment until you hear a click
4. **Check the display** — Albi should show the wine name and details within a few seconds

## Troubleshooting

If Albi doesn't recognize the bag:

- Remove and reinsert the bag
- Make sure the bag is fully seated (pushed all the way in)
- Check that the NFC tag on the bag is not damaged
- Try adding the bag manually via the app
