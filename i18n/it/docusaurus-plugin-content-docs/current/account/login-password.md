# Problemi di Login e Password

## Password Dimenticata?

1. Nella schermata di login, seleziona **"Password Dimenticata"**
2. Inserisci il tuo indirizzo email registrato
3. Controlla la tua casella di posta per il link di reset
4. Crea una nuova password seguendo i criteri richiesti
5. Conferma e accedi

## Non Riesci ad Accedere?

- Assicurati di usare l'**indirizzo email corretto** per il tuo account
- Controlla che il Blocco Maiuscole sia disattivato
- Prova a reimpostare la password (vedi sopra)
- Se ti sei registrato con Google o Apple, usa quei pulsanti di accesso

## Cambiare la Password

1. Vai in **Il tuo Profilo**
2. Seleziona **"Impostazioni e Privacy"**
3. Scegli **"Cambia Password"**
4. Inserisci la password attuale e quella nuova
5. Salva
