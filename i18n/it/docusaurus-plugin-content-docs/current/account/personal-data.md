# Gestire i Tuoi Dati Personali

## Modificare le Informazioni Personali

1. Vai in **Il tuo Profilo**
2. Seleziona **"Impostazioni e Privacy"**
3. Scegli **"Gestione Dati"**

Puoi aggiornare:
- Nome e cognome
- Data di nascita
- Nazionalità
- Genere

Tutte le modifiche vengono salvate automaticamente.

## Cambiare l'Indirizzo Email

1. Vai in **Profilo → Impostazioni e Privacy → Cambia Email**
2. Inserisci il tuo indirizzo email attuale
3. Inserisci il nuovo indirizzo email
4. Conferma e salva

## Cambiare le Impostazioni dell'App

Vai in **Profilo → Impostazioni e Privacy → Gestione Parametri** per modificare:
- Lingua dell'app
- Unità di misura
- Formato data
- Formato orario

## Gestire gli Indirizzi di Spedizione

Vai in **Profilo → Indirizzi di Spedizione** per:
- **Modificare** un indirizzo esistente (selezionalo, modifica, salva)
- **Aggiungere** un nuovo indirizzo (seleziona "Nuovo Indirizzo", compila i campi, salva)
