# Come Funziona?

Albi funziona automaticamente una volta configurato.

1. **Inserisci** una Smart Wine Bag nel dispenser
2. **Riconoscimento automatico** — Albi identifica il tipo di vino, il suo stato di conservazione e le caratteristiche
3. **Controllo della temperatura** — Albi riscalda o raffredda il vino in base al tempo che imposti per raggiungere la temperatura consigliata o preferita
4. **Servi** un bicchiere perfetto quando vuoi

Albi gestisce la conservazione automaticamente, così non dovrai mai preoccuparti che il vino aperto si rovini.
