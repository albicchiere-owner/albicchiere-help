# Impostazioni Dispositivo e Info Tecniche

## Accedere alle Impostazioni Dispositivo

**Se è installata una Wine Bag:**
1. Vai in Esplora
2. Seleziona la card del dispenser
3. Tocca l'**icona impostazioni** (in alto a destra)

**Se non è presente nessuna Wine Bag:**
1. Vai in Esplora
2. Tocca direttamente la card del dispenser

## Cosa Puoi Fare nelle Impostazioni Dispositivo

- Visualizzare le specifiche tecniche
- Rinominare il dispenser
- Cambiare stanza
- Impostare la modalità di raffreddamento
- Gestire il parental control
- Ripristinare le impostazioni di fabbrica

## Informazioni Tecniche

Scorri fino in fondo nelle Impostazioni Dispositivo per trovare:

| Dato | Descrizione |
|------|-------------|
| Numero Modello | Versione hardware |
| Numero Seriale | ID dispositivo univoco |
| Indirizzo MAC | Identificatore di rete |
| Stato Wi-Fi | Stato connessione |
| Versione Firmware | Software attuale |

Queste informazioni sono utili quando contatti il supporto.

## Ripristino di Fabbrica

Vai in **Impostazioni Dispositivo → Ripristina Impostazioni di Fabbrica** e conferma. Questo cancellerà tutte le impostazioni locali sul dispositivo.

:::caution
Il ripristino di fabbrica non può essere annullato. Il dispositivo dovrà essere configurato nuovamente da zero.
:::
