# Come Regolare la Temperatura di Servizio

## Cambiare Temperatura Senza Versare

1. Apri la pagina del dispenser
2. Seleziona il **controllo temperatura**
3. Scorri tra le temperature disponibili
4. Imposta la temperatura preferita o consigliata
5. Conferma la selezione

Albi inizierà a riscaldare o raffreddare per raggiungere la temperatura target.

## Modalità di Raffreddamento

| Modalità | Descrizione | Ideale per |
|----------|-------------|------------|
| **Boost** | Velocità massima ventola, raffreddamento rapido | Quando hai fretta |
| **Eco** | Prestazioni bilanciate, consumo ottimale | Uso quotidiano (default consigliato) |
| **Silent** | Rumore minimo | Notte, ambienti silenziosi |

Per cambiare modalità: vai in **Impostazioni Dispositivo → Modalità Raffreddamento**.
