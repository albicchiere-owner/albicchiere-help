# Come Aggiungere una Smart Wine Bag

## Passaggi

1. Tocca l'**icona del calice** nella schermata principale
2. Seleziona **"Aggiungi Smart Wine Bag"**
3. Albi avvierà il **processo di verifica del vino**

### Se ha successo:
- Tocca **"Aggiungi alla Cantina"** per salvare il vino nella tua collezione

### Se non riesce:
- Controlla il messaggio di errore mostrato
- Assicurati che la bag sia correttamente posizionata in Albi (vedi [Posizionamento della Bag](../smart-dispenser/smart-bag-positioning))
- Riprova o aggiungi il vino manualmente tramite la [Scansione Etichetta](label-scan)
