# Come Programmare una Mescita

Puoi programmare Albi per avere il tuo bicchiere di vino perfetto pronto esattamente quando vuoi.

## Programmare una Mescita

1. Vai alla pagina del dispenser
2. Scorri fino alla **sezione scheduling**
3. Tocca **"Programma"**
4. Scegli l'orario desiderato
5. Conferma

## Modificare o Annullare

Per modificare o annullare una mescita programmata:

1. Vai alla pagina del dispenser
2. Trova la programmazione attiva
3. Tocca per **modificare** (cambia orario) o **annullare** (rimuovi la programmazione)
