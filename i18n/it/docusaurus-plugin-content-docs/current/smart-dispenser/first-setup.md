# Prima Configurazione

Devi configurare Albi una sola volta.

1. Accendi Albi
2. Segui le **istruzioni a schermo** per connettere Albi alla rete Wi-Fi
3. Apri l'app Albicchiere e **associa il dispositivo**
4. Fatto — da ora Albi gestirà tutto automaticamente

:::info
Non dovrai ripetere questa procedura ogni volta che usi Albi. Dopo la configurazione iniziale, si connette automaticamente.
:::
